# Startmine Miro CORS Helper (إضافة تخطي CORS لميرو)

هذه الإضافة مخصصة للسماح لـ Startmine بسحب ورفع صور Miro الأصلية دون مواجهة مشكلات الـ CORS أو أخطاء الأمان الخاصة بالمتصفح، مع الحفاظ على شفافية الصور بالكامل.

## طريقة التثبيت في المتصفح (Google Chrome / Microsoft Edge)

1. افتح المتصفح واذهب إلى صفحة الإضافات:
   - في جوجل كروم: اكتب `chrome://extensions` في شريط العنوان واضغط Enter.
   - في مايكروسوفت إيدج: اكتب `edge://extensions` في شريط العنوان واضغط Enter.

2. قم بتفعيل **وضع المطور (Developer Mode)** في الزاوية العلوية اليمنى (أو اليسرى حسب لغة المتصفح).

3. اضغط على زر **تحميل حزمة غير مضغوطة (Load unpacked)** الذي سيظهر في الزاوية العلوية اليسرى.

4. اختر المجلد الخاص بالإضافة من جهازك:
   - مسار المجلد: `C:\Users\NTRA\OneDrive - NTRA-EG\17May_Startmine\miro-cors-helper`

5. بمجرد اختيار المجلد، ستظهر لك إضافة باسم **Startmine Miro CORS Helper** مفعلة وجاهزة للعمل!

## كيف تعمل؟
الإضافة تقوم بمراقبة الطلبات الصادرة من Startmine إلى موقع ميرو، وتضيف ترويسات الأمان اللازمة (`Access-Control-Allow-Origin` و `Access-Control-Allow-Credentials`) ديناميكياً للطلب لكي يسمح المتصفح لجافاسكريبت بقراءة وتحميل الصورة الأصلية الشفافة ورفعها لـ ImgBB.
